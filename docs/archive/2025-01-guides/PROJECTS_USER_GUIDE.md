# Projects Feature - User Guide

## What Are Projects?

Projects in KimbleAI help you organize conversations by topic, client, or category - just like ChatGPT's project feature.

## FAQ

**Q: How do I move an existing conversation to a project?**
A: Currently, you need to manually assign it. We're working on a "Assign to Project" feature.

**Q: Can I have conversations in multiple projects?**
A: Each conversation belongs to exactly one project (or "General" if unassigned).

**Q: What happens to conversations if I delete a project?**
A: They become unassigned and move to "General" section.

**Q: Can I see all my projects at once?**
A: Yes, they're all listed in the "PROJECTS" section of the sidebar.

**Q: How many projects can I create?**
A: Unlimited! Create as many as you need.

## Getting Help

If projects aren't working:
1. Check that you selected a project BEFORE clicking "New chat"
2. Verify the project appears highlighted in sidebar
3. Look for the project header in your chat
4. Check the sidebar for project grouping

Still having issues? The system may need a quick refresh:
1. Refresh the page (Ctrl+R or Cmd+R)
2. Try creating a test project
3. Try selecting it and starting a new chat
